linux命令

df （Disk free）

df 命令用于显示目前在 Linux 系统上的文件系统的磁盘使用情况统计

-h = --human-readable 使用人类可读的格式 (预设值是不加这个选项的…)

free

free 命令用于显示内存状态。

free 指令会显示内存的使用情况，包括实体内存，虚拟的交换文件内存，共享内存区段，以及系统核心使用的缓冲区等。

-m = --mega 以 MB 为单位显示内存使用情况

-g = --giga 以 GB 为单位显示内存使用情况

-h = --human 　以合适的单位显示内存使用情况，最大为三位数，自动计算对应的单位值

top

Linux top 命令用于实时显示负载的动态。

使用权限：所有使用者。

tips

1.负载均衡的数值不能超过 10

2.如果某服务长期占用 cpu 或者 men，去检查这个进程是在做什么

3.如果 cpu 飙升 3000% 以上，夯住 或者有问题。代码级别的问题，如果不是自己编写的代码，大概率硬件级别 --> 如果重启，重启不来，说明内存条坏了

ps（Process Status）

ps 命令用于显示当前进程 (process) 的状态。

-ef 显示所有命令，连带命令行，常常和 grep 配合使用

查看关于ssh的进程，同时过滤掉自己

netstat（net status）

netstat 命令用于显示网络状态。

利用 netstat 指令可让你得知整个 Linux 系统的网络情况。

-n = --numeric 直接使用 IP 地址，而不通过域名服务器

-l = --listening 显示监控中的服务器的 Socket。

-p = --programs 显示正在使用 Socket 的程序识别码和程序名称。

常用组合：查看进程得到的 pid 号查看端口号

tips

1.如果查询出来的端口号前面的 ip 是 127.0.0.1 或者 localhost，属于本地回环 ip 地址，需要修改相应的配置文件

2.在 centos 部署大数据组件，发现一个错误 Connection refused 解决思路：ping ip 测试 ip->telnet ip port 测试 ip 和端口号 ->防火墙

ifconfig

Linux ifconfig 命令用于显示或设置网络设备。

ifconfig 可设置网络设备的状态，或是显示目前的设置
