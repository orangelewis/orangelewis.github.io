linux命令

useradd

‘’‘
useradd ruoze
’‘’
创建一个普通用户，默认创建这个名称的用户组ruoze,
且设置这个用户 主组为ruoze，且创建/home/ruoze
帐号建好之后，再用 passwd 设定帐号的密码。而可用 userdel 删除帐号。使用 useradd 指令所建立的帐号，实际上是保存在 /etc/passwd 文本文件中。
所以修改 passwd 文件 ，可能会导致用户无法登录

userdel

userdel 可删除用户帐号与相关的文件。若不加参数，则仅删除用户帐号，而不删除相关文件。
如果当前用户所在的组只有一个用户（它本身），当这个用户删除时，组会校验就他自己，会自动删除
-r = --remove 删除用户登入目录以及目录中所有文件。

su

Switch user
su 命令用于变更为其他使用者的身份，除 root 外，需要键入该使用者的密码。
在账号名称前加-，改变目录在该用户的家目录
如
变更帐号为 clsung 并改变工作目录至 clsung 的家目录（home dir），且执行环境变量文件
’’‘
su - clsung
’’’

groupadd

groupadd 命令用于创建一个新的工作组，新工作组的信息将被添加到系统文件中。

usermod

usermod 命令用于修改用户帐号。
usermod 可用来修改用户帐号的各项设定。
-a = --append 追加
-G = --group 修改用户所属的附加群组
-g = --gid 修改用户所属的群组(主组)。

用户添加组

usermod -a -G bigdata ruoze

修改用户的主组

usermod --gid bigdata ruoze

du

du = disk usage
显示文件或目录所占用的磁盘空间。
-h = --human-readable 输出文件系统分区使用的情况，例如：10KB，10MB，10GB 等
-s = --summarize 显示文件或整个目录的大小，默认单位是 KB

find

find 命令用来在指定目录下查找文件。任何位于参数之前的字符串都将被视为欲查找的目录名。如果使用该命令时，不设置任何参数，则 find 命令将在当前目录下查找子目录与文件。并且将查找到的子目录和文件全部进行显示。
-name name, -iname name : 文件名称符合 name 的文件。iname 会忽略大小写

ps

ps=Process Status 进程状态
ps -ef //显示所有命令，连带命令行

top

命令用于实时显示 process 的动态。
使用权限：所有使用者。

ll要关注的东西

权限，所属用户和用户组，时间，大小（文件）

用户和用户组

文件位置

用户文件：/usr/sbin/user*
用户组文件：/usr/sbin/group*

恢复用户样式

模拟删除
rm -rf .bash*
1.从skel拷贝到用户目录
2.修改权限

cp /etc/skel/.* /home/ruoze/
chown ruoze:ruoze .bash*

用户获得最大权限

添加普通用户具备 root 权限的文件是：/etc/sudoers
vi /etc/sudoers
ruoze ALL=(root) NOPASSWD:ALL

权限

chmod 命令用来修改文件或者目录的读写执行权限，加 -R 表示递归修改
chown 命令用来修改文件或者目录的属主和属组，加 -R 表示递归修改
如
chmod -R 777 文件夹/文件路径
chown -R 用户:用户组 文件夹/文件路径

数字的含义

ll test.txt
-rw-r–r-- 1 root root 12 Nov 12 23:36 test.txt
r: read 4
w: write 2
x: 执行 1
-: 没权限 0
7=rwx
3=wx
5=rx
　rw - 第一组 6 代表文件或文件夹的用户 root，读写
　r-- 第二组 4 代表文件或文件夹的用户组 root，读
　r-- 第三组 4 代表其他组的所属用户对这个文件或文件夹的权限：读
